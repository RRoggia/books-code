# Cheat Sheet

## Iteração

### Contar um numero de vezes (Crescente)
```javascript
let numeroDeVezes = 100
for(let i =0; i <= numeroDeVezes; i++){
	console.log(i)
}
```

### Contar um numero de vezes (Decrescente)
```javascript
let numeroDeVezes = 100
for(let i = numeroDeVezes; numeroDeVezes >= 0; i--){
	console.log(i)
}
```
